# Ali Haroon | Frontend Portfolio & JS Lab 🚀

Welcome to my personal frontend showcase. This repository is a collection of high-performance web interfaces and logic experiments built during my 6th semester of BSCS at COMSATS.

## 📌 The Mission
As a **Full Stack Web Developer**, I believe that a solid backend is only as good as the interface that communicates with it. This portfolio focuses on **Core JavaScript Mastery**, **Modern CSS Layouts**, and **UI Performance**—the essentials of a great user experience.

## 🛠️ Tech Stack (Frontend Focus)
* **Languages:** HTML5, CSS3, JavaScript (ES6+)
* **Styling:** CSS Flexbox, Variables, and Animations
* **Logic:** Vanilla JS (No frameworks used to ensure maximum performance)
* **Tools:** VS Code, Git, Chrome DevTools

## 🧪 Featured Lab: JavaScript 30
These projects demonstrate my ability to solve complex problems using only native browser APIs. I am Following wes bos JS30 series And doing project then add new functionalities to it.

* **[CPU Load Monitor](https://github.com/AliHaroon111/JavaScript-Portfolio/tree/main/JavaScript_30/02%20-%20JS%20and%20CSS%20Clock/mastery-task):** A dynamic gauge that uses math mapping to translate data into degrees and colors.

## ⚙️ Architecture Highlights
* **Clean Separation of Concerns:** Logic (JS) and Design (CSS) are maintained in separate, optimized files.
* **Responsive Design:** Built with a "mobile-first" mindset using Flexbox.
* **Dynamic Content:** Interactive elements that respond to user input and time-based events.
